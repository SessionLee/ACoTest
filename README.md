# Huge Number Calculator

A multi-language calculator that supports adding extremely large integers (50+ digits).

## Features

- Supports integers with 50+ digits
- Available in Python and Go

## Usage

### Python

```bash
python big_add.py 123456789012345678901234567890 + 987654321098765432109876543210
```

### Go

```bash
go run cmd/go/big_add.go 123456789012345678901234567890 + 987654321098765432109876543210
```
